--------------------------------------------------------------------------------
TOOLS TO INSTALL FOR PRACTICE
--------------------------------------------------------------------------------

  kubectl       — The Kubernetes CLI (required for everything)
  minikube      — Single-node K8s cluster on your laptop (perfect for learning)
  Docker Desktop — Container runtime with built-in Kubernetes option
  Lens / k9s    — GUI tools for visualising Kubernetes clusters

  Quick start:
    1. Install minikube
    2. Run: minikube start
    3. Follow File 04 to create your first Pod
