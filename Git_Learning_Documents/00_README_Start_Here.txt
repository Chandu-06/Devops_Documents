RECOMMENDED READING ORDER
----------------------------------------------------------------

If you are completely new:
    01 -> 02 -> 03 -> 04 -> 05 -> 06 -> 07 -> 08 -> 13

If you know the basics and want to go deeper:
    08 (Git Flow) -> 09 (Rebase) -> 10 (Tags) -> 11 (gitignore)

    13 (Day 1 Setup Guide) first, then read the rest
